# Waterfront Pathways GTFS Sample

This fixture is a compact GTFS Pathways excerpt from MobilityData's published
Waterfront Station example, with `level_id` and `levels.txt` added so Chiyoda
can build strict multi-floor `layout.floors`.

Source:
<https://raw.githubusercontent.com/MobilityData/gtfs.org/main/docs/en/documentation/schedule/examples/pathways.md>
